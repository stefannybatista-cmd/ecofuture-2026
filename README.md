 EcoFuture 2026 — Projeto Agrinho

**EcoFuture 2026** é um projeto educacional interativo desenvolvido para o concurso **Agrinho**, que explora a evolução da agricultura rumo a um futuro regenerativo.

Funcionalidades

- **Chat com IA Gaia**: inteligência artificial educativa que responde perguntas sobre bioinovação.
- **Linha do Tempo BioInovadora**: anos de 2023 a 2100 mostrando tecnologias regenerativas.
- **Galeria interativa** com imagens temáticas e navegação.
- **Métricas de impacto** animadas com barras de progresso.
- **Perguntas rápidas** e sugestões de conversa.
- **Easter eggs** escondidos (comandos secretos no chat e console).
- Modo responsivo (desktop e mobile).

 Tecnologias utilizadas

- HTML5 semântico
- CSS3 customizado + Tailwind CSS
- JavaScript vanilla (sem frameworks)
- Font Awesome 6
- Google Fonts (Orbitron + Inter)
